# 预期差：美股的短线博弈 (Earnings Surprise)

在美股市场，股价的短期波动往往不取决于绝对的好坏，而取决于**实际结果与市场预期之间的差距**。

## 1. "Priced In" (已计入预期)
为什么有时好消息公布后股价反而下跌？因为市场是前瞻的，好消息可能已经被“提前计入”了股价。

## 2. 考试成绩比喻 (The Exam Grade Analogy)
把财报季看作是学生拿成绩单：

*   **学生 A (如巅峰期的 NVDA)**:
    *   **市场预期**: 考 98 分 (完美无瑕)。
    *   **实际成绩**: 考 95 分 (非常优秀)。
    *   **结果**: **失望 (Disappointment) -> 股价下跌**。因为市场定价是基于完美的预期，任何瑕疵都会被放大。

*   **学生 B (如困境反转中的 Intel)**:
    *   **市场预期**: 考 50 分 (不及格)。
    *   **实际成绩**: 考 60 分 (勉强及格)。
    *   **结果**: **惊喜 (Surprise) -> 股价大涨**。因为只要比“烂透了”好一点，就超出了预期。

## 3. Whisper Numbers vs Consensus (私下预期 vs 官方共识)
*   **Consensus**: 分析师公开预测的平均值。
*   **Whisper Numbers**: 交易员和买方机构在私下里的真实期望值。往往比 Consensus 更高（对于热门股）。
*   **风险**: 仅仅击败 Consensus 是不够的，必须击败 Whisper Numbers 才能推动股价上涨。

## 4. 指引 (Guidance) 是关键
美股极其看重未来。
*   即使上个季度的业绩超预期（Beat），如果公司给出的下季度**指引 (Guidance)** 不及预期（Miss），股价依然会暴跌。
*   **逻辑**: 过去已成历史，股市买的是未来现金流。
