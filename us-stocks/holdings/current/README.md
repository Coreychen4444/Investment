# Current Holdings Notes

These files are working notes (not broker exports) for each holding:
- Role in the portfolio (core vs. satellite vs. high-beta)
- Thesis strength vs. sentiment beta
- What matters now
- Add / trim / exit logic
- Danger signs

Use them to separate:
- Core expressions (highest conviction, hold through noise)
- Secondary expressions (follow-through vehicles)
- High-beta sentiment names (tighter risk management)
- Steadier quality names (lower drama, strategic exposure)

## How to use
Create one `.md` file per holding using the example template (`_EXAMPLE_.md`).
Update regularly as thesis evolves.
