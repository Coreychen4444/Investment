# Polymarket Workspace

这个目录专门用于 **Polymarket / Valorant 比赛相关交易**。

目标：
- 与现有美股投资体系分开
- 单独记录 PM 交易策略、交易日记、复盘、经验教训
- 保留底层共通纪律，但不混淆市场逻辑

## 目录结构
- `strategy/` —— 交易策略、框架、赔率理解、盘口打法
- `journal/` —— 日常交易记录
- `reviews/` —— 复盘、错误分析、经验教训
- `knowledge/` —— Valorant 比赛理解、战队/地图池/版本环境等知识
- `templates/` —— 固定模板

## 原则
- PM 交易不和美股持仓文件混写
- Valorant 交易视为一套独立策略系统
- 若有共通纪律（仓位、止损、预期差、执行规则），可在复盘中引用，但不要混成同一套 market narrative
