# PM Live Iron Rules

## Rule 1 — One build, one add
同一场 live 盘口、同一方向：
- 只允许 **1 次建仓**
- 最多 **1 次补仓**
- **禁止第 3 次加仓**

## Rule 2 — Trade price repair, not miracle comeback
PM live 盘交易的是 **赔率修复**，不是赌最终翻盘。
如果赔率已经修复，就应优先考虑止盈，而不是幻想拿满终局。

## Rule 3 — Thesis can fail in real time
最初的错价假设不是永久有效。
如果比赛内容持续恶化，并不断否定原始逻辑，必须承认判断失效。

## Rule 4 — Hard loss cap per match
单场亏损必须有硬上限。
禁止一场回吐整个阶段盈利。

## Rule 5 — Soft-info edge is not a license to oversize
懂 Valorant、懂软信息，不等于可以不断加仓。
edge 只能给你入场优势，不能替代仓位纪律。
